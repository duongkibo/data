## Camera Move Vocabulary

| Move | Description | Use case |
|---|---|---|
| PUSH | Camera moves forward into scene | Opening anchor shots, betrayal approach |
| PULL | Camera moves backward revealing scene | Isolation reveals, epilogue payoffs |
| PAN-L/R | Camera sweeps left or right | Scene establishing, crowd shots |
| TILT-UP/DN | Camera tilts up or down | Document reveals, height reveals |
| ORBIT-L/R | Camera circles subject | Character moments, build tension |
| RACK-FOCUS | Shifts focus from background to foreground | Object reveals, foreground detail |
| DOLLY | Tracks forward with subject | Following character movement |
| CRANE-UP | Camera rises vertically | Final reveal shots, epilogue |
| DUTCH | Tilted camera angle — psychological wrongness | Beat 3-5 B/C ONLY — NEVER Epilogue |
| DESCENT | Camera slowly moves downward | Document highlights, floor-level |
| STATIC | No movement — hold the frame | Peak text/betrayal moments |
