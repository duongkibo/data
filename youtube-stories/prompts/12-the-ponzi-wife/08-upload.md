## 24-hour performance check

| Metric | Target | Action if below |
|---|---|---|
| CTR | 5%+ | Swap to V2 thumbnail |
| Avg view duration | 6+ min (52%) | Check Analytics drop-off |
| Likes/views ratio | 4%+ | Check comment sentiment |
| Comments per 1k views | 8+ | Add poll card if missing |
| Subs gained per 100 views | 2+ | Check end screen performance |

---

## Rollback plan

If CTR < 3% at 48h:
1. Swap thumbnail V1 → V2 → monitor 24h
2. If still < 3% → swap to V3
3. Consider A/B testing title V3 or V4 from `07-title-description.md`
4. Log result in `analytics/EP12-review.md`
