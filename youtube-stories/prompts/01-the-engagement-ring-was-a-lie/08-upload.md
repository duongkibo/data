# 08 — Upload Protocol

> Final QC + upload YouTube Studio + first-hour engagement

---

## 🎬 Edit workflow (CapCut, trước khi upload)

1. **Import voice** 3 chunks → merge sequential trên Track 1 Audio
2. **Import 40 video clips** (C01→C40) lên Track 1 Video theo order
3. **Align với voice** — mỗi clip match 1 beat trong script
4. **Insert 15 Ken Burns stills** vào gap giữa clips:
   - Scale keyframe 100% → 115% over 15-30s mỗi still
5. **Background music** CapCut library:
   - Genre: Emotional cinematic / Ambient piano
   - Volume: **-18dB** (dưới voice -6dB)
6. **Color grade** (optional):
   - Contrast +5, Saturation -3
   - Warm scenes: Temp +2
   - Cold scenes: Temp -3
7. **Subtitle**: Auto-generate English → manual fix Sarah/Daniel/Patricia/Portland
8. **Export**:
   - 1920×1080
   - 30fps
   - H.264
   - 10 Mbps
   - MP4

**Save**: `media/final/episode-01-final.mp4`

## 📤 YouTube Studio upload steps

### 1. Upload file
YouTube Studio → Create → Upload videos → chọn `episode-01-final.mp4`

### 2. Details tab
- **Title**: paste từ `07-title-description.md`
- **Description**: paste từ `07-title-description.md`
- **Thumbnail**: upload `v1.png` (primary) — lưu v2, v3 cho A/B test sau 1k subs
- **Playlists**: Tạo mới "Karma Caught Up Stories" → add
- **Audience**: **No, it's not made for kids**
- **Age restriction**: No
- More options:
  - Tags: paste 15 tags từ `07-title-description.md`
  - Language: English
  - Category: **People & Blogs** hoặc **Entertainment**
  - License: Standard YouTube License
  - ✅ Allow embedding
  - ✅ Publish to subscriptions feed + notify
  - Shorts sampling: OFF

### 3. Video elements tab
- **Subtitles**: Upload SRT từ CapCut export (English)
- **End screen**:
  - Subscribe button (bottom-right)
  - Next video (bottom-left, auto "best for viewer")
- **Cards**:
  - 1 card at **5:30** (pivot moment) → link to channel subscribe

### 4. Checks tab
- Copyright scan: wait green ✅
- Ad suitability: check Yellow/Green — nếu Yellow request review

### 5. Visibility tab
- **Schedule**: Tuesday 9:00 PM EST (next available) hoặc Friday 9:00 PM EST
- ✅ Set as instant Premiere (optional — tạo hype)

→ Click **Schedule**

## 📊 Post-24h review

Check YouTube Studio Analytics:
- **CTR**: target 4-8%
- **Avg view duration**: target 5-7 phút
- **Watch time**: target 3k-15k hours
- **Subscribers gained**: target 50-500
- **Top traffic source**: Browse features + Suggested (good sign)

**Action based on data**:
| Metric | Below target | Action |
|---|---|---|
| CTR | <3% | Swap thumbnail → V2/V3 |
| AVD | <4 min | Hook fail, adjust next script |
| Watch time | <1k hours | Cross-post Reddit + Twitter |
| Subs | <20 | CTA stronger trong next video |
