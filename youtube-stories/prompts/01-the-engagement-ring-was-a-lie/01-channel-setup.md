## ✅ Upload vào YouTube Studio

1. Customization → **Branding**:
   - Profile picture → upload `avatar.png`
   - Banner image → upload `banner.png`
2. Customization → **Basic info**:
   - Name → `Karma Caught Up`
   - Handle → `@KarmaCaughtUp`
   - Description → paste từ trên
3. Customization → **Layout** → (để mặc định, setup sau khi có video)
4. Settings → **Channel** → Country: Vietnam hoặc US (US tốt hơn cho RPM)
