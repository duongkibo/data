## Camera Move Library

| Move | Usage |
|---|---|
| PUSH | Slow zoom toward subject — tension, intimacy |
| PULL | Reveal more of scene — isolation, scale |
| PAN-L / PAN-R | Horizontal reveal — survey, passage of time |
| TILT-UP / TILT-DN | Vertical reveal — scale, importance |
| ORBIT-L / ORBIT-R | Circle around subject — emphasis, celebration |
| RACK-FOCUS | Shift focus between planes — perspective change |
| DOLLY | Physical move toward/away — approach, departure |
| CRANE-UP | Rise above — overview, conclusion |
| DUTCH | Tilted frame — psychological unease (**BEAT 3-5 ONLY**) |
| DESCENT | Bird's eye down — omniscient, surveillance |
| STATIC | No camera move — weight, silence, stasis |
