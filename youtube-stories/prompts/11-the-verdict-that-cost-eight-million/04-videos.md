## Setup Freepik Kling 2.5 Turbo

- Model: **Kling 2.5 Turbo**
- Ratio: **16:9**
- Duration: **10s** (max)
- Resolution: **1080p**
- Unlimited credits
- **Save**: `media/videos/C01.mp4` -> `C40.mp4`

## Workflow per clip

1. Open Kling 2.5 Turbo
2. **Upload start frame** `F{NN}.png` as start image
3. **Paste motion prompt** below (1 action only)
4. Duration **10s**
5. Generate -> save `C{NN}.mp4`
6. If drift/fail -> regen (unlimited)

## Rules (critical)

- **1 action per clip** — NO multi-shot, NO scene transitions. Kling drifts on complex prompts.
- Camera motion explicit: "slow dolly in", "gentle pan left", "hand moving slowly"
- Keep 10 seconds continuous — no cuts
- Keep character proportions matching start frame
