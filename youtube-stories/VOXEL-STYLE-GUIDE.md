## 🎯 What is Voxel Art Original?

**Voxel = Volume + Pixel** = 3D cubes that build worlds.

### Visual reference
- **Astroneer** (game) — smooth pastel voxel exploration
- **Trove** — colorful adventure voxel
- **Cube World** — vibrant voxel RPG
- **Crossy Road** — clean voxel mobile game
- **Sandship** — premium voxel industrial
- **Premium voxel artists**: Sir_carma, mineways, pjmuts (Twitter/ArtStation)

### NOT Minecraft because
- ❌ Raw Minecraft = pixelated 16x16 blocky textures
- ✅ Voxel art original = smooth shading, premium lighting, cinematic atmosphere
- ✅ Bigger blocks possible (less granular)
- ✅ Clean colors, no Minecraft branding/items

## 🎨 Visual identity

### Color palette (locked)
```
PRIMARY DARK (mood):
- Deep Navy:        #0B1426
- Charcoal:         #1F2937  
- Plum:             #4C1D95

ACCENT (single per scene):
- Warm Gold:        #D4A547
- Sunset Orange:    #F59E0B
- Burgundy:         #8B1E2F
- Sage Green:       #6B7F5A
- Cream:            #F3E9D2

VOXEL HIGHLIGHTS:
- Block edge highlight: lighter shade +15%
- Block shadow: darker shade -20%
- Smooth gradient between cubes
```

### Lighting style
- **Single dramatic source** (sun, lamp, screen)
- **Long voxel shadows** (low angle creates drama)
- **Volumetric god rays** between cube structures
- **Color temperature shift per beat**
- **Cube edge highlights** for definition

### Composition rules
- **Isometric perspective** (45° classic voxel angle)
- **OR cinematic camera** (low angle 3D)
- **Negative space** (empty quiet cube areas)
- **Single focal point** per frame
- **Cube architecture** that tells story

### Camera angles
1. **Isometric**: Classic voxel game view (45° down) — best for environments
2. **Low cinematic**: Eye-level 3D — best for emotional close-ups
3. **Top-down**: Bird's eye — for transitions
4. **First person**: Through character eyes — for POV moments
